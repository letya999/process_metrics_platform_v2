## ❌ Что НЕ НУЖНО для self-hosted

| Enterprise-решение | Почему не нужно |
|-------------------|-----------------|
| Kubernetes | 1 droplet достаточно |
| Redis кластер | Для 1-5 пользователей не критично |
| PostgreSQL replicas | Self-hosted не требует HA |
| HashiCorp Vault | Fernet или env-переменные |
| JWT + RBAC | Basic Auth + IP whitelist |
| Prometheus + Grafana | Dagster UI имеет мониторинг |
| PagerDuty/Slack alerts | Смотреть логи руками |
| WAF | Cloudflare free tier достаточно |
